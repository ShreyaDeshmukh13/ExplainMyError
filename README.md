# 🚀 ExplainMyError

ExplainMyError is a full-stack debugging assistant that helps developers understand programming errors by generating simplified explanations, identifying root causes, and suggesting fixes.

---

## 🎯 Features

* 🧠 **Error Analysis** – Converts complex error messages into simple explanations
* 🏷 **Smart Categorization** – Automatically classifies errors (TypeError, IndexError, etc.)
* 🔍 **Search & Filter** – Quickly find past errors by keyword or category
* 📊 **Error Analytics** – View total errors and most common error types
* 🕘 **Recent Errors** – Displays latest analyzed errors
* ✏️ **CRUD Functionality** – Create, update, and delete error logs
* 📋 **Copy Fix** – Easily copy suggested fixes

---

## 🧱 Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite (SQLAlchemy)
* **Frontend:** HTML, CSS, JavaScript
* **UI Design:** Custom light theme

## 📌 Usage

1. Enter a programming error
2. Click **Analyze**
3. View explanation, cause, and fix
4. Browse history or edit past errors
